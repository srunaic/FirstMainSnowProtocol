# 3D MetaVerse Park
동성로의 국체보상공원을 모티브로한 메타버스 시티 입니다.

## 🖥️ 프로젝트 소개
단순히 돌아다니는 것만이 아닌 오락거리 먹거리 홍보영상등
메타버스에서 즐길수 있는 콘텐츠등을 제공합니다.
